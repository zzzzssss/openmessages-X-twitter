Up 
get /twitterActivities
Get all the twitters info including hashtags (twitterActivitiesGet)

Query parameters

id (optional)
Query Parameter — twitter activity id in activities table
hashtag (optional)
Query Parameter — hashtag of the twitter

Return type

Activitiesfull

Example data

Content-Type: application/json
{
  "urls" : "urls",
  "user_id" : "user_id",
  "activity_id" : "activity_id",
  "created_at" : "created_at",
  "id" : 0,
  "body" : "body",
  "hash_tags" : "hash_tags"
}

Produces

  • application/json

Responses

200

successful operation Activitiesfull

400

Invalid id supplied
